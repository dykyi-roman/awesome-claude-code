### Immediate Actions (Critical)

1. **Separate declarations from side effects**
   - Move configuration to bootstrap files
   - Create dedicated initialization scripts
   - See: `references/psr-1-basic.md`

2. **Fix class naming conventions**
   ```bash
   php-cs-fixer fix src/ --rules=class_definition
   ```

### Short-term Actions (Warning)

1. **Configure editor settings**
   ```json
   // .editorconfig
   [*.php]
   indent_style = space
   indent_size = 4
   end_of_line = lf
   charset = utf-8
   trim_trailing_whitespace = true
   insert_final_newline = true
   ```

2. **Add pre-commit hooks**
